Entendendo formulários e com métodos GET e POST

Código utilizado em PHP para acessar os Arrays do método GET E POST 
    GET: variável $_GET
    POST: variável : $_POST

    Exemplo de uso:

    <?php
    echo 'GET:';
    print_r($_GET);
    echo 'POST:';
    print_r($_POST);
